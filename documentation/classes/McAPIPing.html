fetch()

fetch(String|\McAPIVersion  $version = null, boolean  $userVersion = false) : boolean|void

Fetching information from the given Minecraft server

Parameters

String|\McAPIVersion $version

The Minecraft-Version depends on the used version on the request server. If it's null the script tries to calculate the version, but this can takes a while.
boolean $userVersion

If it true the script tries to read the given version

Returns

boolean|void —

If it true the fetch worked successfully and it returns data
